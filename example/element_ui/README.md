# element_ui + wangeditor5-for-vue2

`wangeditor5-for-vue2` 的 `element_ui` 使用示例

## 运行

### yarn

```sh
// 安装依赖
yarn

// 安装 wangeditor5-for-vue2
yarn add wangeditor5-for-vue2

// 运行
yarn run serve
```

### npm

```sh
// 安装依赖
npm i

// 安装 wangeditor5-for-vue2
npm i wangeditor5-for-vue2

// 运行
npm run serve
```
